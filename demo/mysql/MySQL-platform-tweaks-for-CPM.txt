Download the >> 32-bit << MSI installer package from here:
	http://dev.mysql.com/downloads/connector/odbc/
>> Make sure it is the 32-bit, not 64-bit driver installer. <<

Run installer
cd to \Program Files\MySQL
To Register Unicode driver for Windows, run:
>> myodbc-installer -a -d -n "MySQL ODBC 8.0 Unicode Driver" -t "DRIVER=myodbc8w.dll;SETUP=myodbc8S.dll"

In PVWA, edit MySQL platform:
  Automatic Password Management -> Additional Policy Settings
  Edit the Driver=[name] in ConnectionCommand to match name used above:
  >> Driver=MySQL ODBC 8.0 Unicode Driver
